GLITCH

Select Difficulty

Règles du jeu :
Vous êtes le Glitch (jaune). Déplacez-vous avec WASD ou les flèches.
Consommez toutes les entités (grises, rouges, blanches) pour survivre.
Les Chasseurs (blancs) vous traquent.
Ne laissez pas votre énergie s'épuiser.

WASD/Arrows: Move | CLICK: Spawn Food | SHIFT+CLICK: Spawn Seeker (Red) | CTRL+CLICK: Spawn Hunter (White)
