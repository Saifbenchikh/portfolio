Architecture Modulaire

Philosophie : "Less is More"

À l'heure où le web est saturé de frameworks lourds (React, Angular) pour de simples sites vitrines, j'ai fait le choix radical de la légèreté. Ce portfolio pèse moins de 200ko. Chaque ligne de CSS et de JS a été écrite à la main pour garantir une maîtrise totale du DOM et des performances optimales.

CSS Architecture

Utilisation des Custom Properties (Variables) pour un theming dynamique. Création de motifs abstraits (gradients CSS) pour éviter le chargement d'images lourdes en background.

JavaScript Optimisé

Pas de jQuery. Utilisation de l'API moderne IntersectionObserver pour déclencher les animations au scroll uniquement lorsque l'élément entre dans le viewport.

Performance SEO

Structure HTML5 sémantique (main, article, nav) pour l'accessibilité. Chargement différé (Lazy Loading) des assets non critiques.

Focus Code : Génération de Motifs CSS

CSS3

Au lieu d'importer des fichiers PNG/SVG, les motifs visuels des cartes projets sont générés mathématiquement par le navigateur via des gradients radiaux. Résultat : 0 temps de chargement réseau. 

/* Motif "Data Points" généré en CSS pur */
.abstract-web {
    background-color: #f0f9ff;
    background-image: 
        radial-gradient(circle at center, #bae6fd 1px, transparent 1px),
        radial-gradient(circle at center, #3b82f6 1px, transparent 1px);
    background-size: 20px 20px, 40px 40px; /* Grille mathématique */
    background-position: 0 0, 10px 10px;    /* Décalage pour l'effet de profondeur */
}
